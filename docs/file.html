Les files !

Instructions

La file est composée d'éléments posés les uns à la suite des autres, à la queue-leu-leu. Elle propose deux fonctionnalités :
  • Ajouter un élément à la fin de la file, on dit enfiler. Pour enfiler un élément, cliquez sur lui
  • Défiler, c'est-à-dire enlever l'élément qui est au début de la file.
Dans ce document, nous lirons les files de gauche à droite : le premier élément est à gauche et le dernier à droite.
