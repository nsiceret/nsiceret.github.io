Les files !

Instructions

La file est composée d'éléments posés les uns à la suite des autres, à la queue-leu-leu. Elle propose trois fonctionnalités.
  • Vider la file consiste à enlever tous les éléments de la file, s'il y en a. On obtient une file vide.
  • Ajouter un élément à la fin de la file, on dit enfiler.
  • Défiler, c'est-à-dire enlever l'élément qui est au début de la file.
Vous pouvez créer ou modifier votre propre file en cliquant sur les boutons ci-dessous.
Dans ce document, nous lirons les files de gauche à droite : le premier élément est à gauche et le dernier à droite.
